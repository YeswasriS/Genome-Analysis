# Genome Analysis — Enterococcus faecium E745

## Project
Re-analysis of Zhang et al. 2017 to identify genes 
that help Enterococcus faecium survive in human blood.

## Paper
Zhang et al. (2017). RNA-seq and Tn-seq reveal fitness 
determinants of vancomycin-resistant Enterococcus 
faecium during growth in human serum. 
BMC Genomics 18:893.

## Student
Yeswasri S — Uppsala University 2026

## Repository Structure
- `scripts/` — All analysis scripts
- `assembly/` — Canu assembly + QUAST results
- `annotation/` — Prokka annotation results
- `mapping/` — BWA mapping results
- `quality_control/` — FastQC + Trimmomatic results
- `expression_analysis/` — DESeq2 results
- `results/` — HTSeq count tables
